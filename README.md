# smr-selector
Selection of Supervised Learning-based Sparse Matrix Reordering Algorithms

## pymumps Installation

`pip install --upgrade pip setuptools wheel packaging scikit-build-core`

`sudo apt update`

`sudo apt install -y gfortran libopenmpi-dev openmpi-bin libmumps-dev cmake build-essential`

`pip install pymumps`

You can also verify your packaging version with:

`python3 -c "import packaging; print(packaging.__version__)"`

A modern version (23.x or 24.x+) should work.
